Интересные сессии TechEd 2020

17.11.2020 от Astrafox·Нет комментариев

На TechEd 2020 ожидается довольно много сессий посвященных ABAP разработке и будущему ABAP платформы, наиболее интересные из них:

  • DEV825 Road Map: ABAP — Будут обозначены основные направления в развитии языка и его окружения.
  • DEV260 Build SAP Fiori Apps with the ABAP RESTful Application Programming Model — RAP — фреймворк, заменяющий BOPF, способствующий более ускоренному и простому созданию Backend Odata (и не только Odata) сервисов.
  • DEV102  Major Updates on ABAP RESTful Application Programming Model — обзорная сессия посвящённая нововведениям в RAP.
  • DEV300 SAP Cloud Platform, ABAP Environment: Overview — Обзор ABAP Cloud Environment (aka Steampunk).
  • DEV208 DevOps for ABAP? Try the Git-Enabled Change and Transport System — Возможно одна из самых интересных сессий, Git в мире ABAP разработки набирает обороты, любопытно посмотреть каким образом SAP планирует интегрировать Git и существующую систему контроля версий CTS.
